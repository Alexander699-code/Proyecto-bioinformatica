Caracterización de Genes y Mutaciones Asociadas a Resistencia Antimicrobiana en una Cepa Clínica de Klebsiella pneumoniae Aislada en Lima

Objetivo general:

Detección de Genes AMR y Mutaciones en una Cepa de Klebsiella pneumoniae MDR aislada de muestra clinica de Lima, Perú

Objetivos Específicos

Ensamblar y anotar el genoma bacteriano a partir de lecturas Illumina utilizando herramientas de ensamblaje y anotación como SPAdes y Prokka.

Identificar genes de resistencia antimicrobiana presentes en el genoma mediante herramientas como ABRicate y AMRFinderPlus.

Detectar mutaciones puntuales o variaciones en genes previamente asociados a resistencia.

(Opcional / Complementario) Visualizar el genoma anotado de manera gráfica para facilitar su interpretación.
